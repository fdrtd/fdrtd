# how to use

1. install the fdrtd core middleware
 
```shell
pip install fdrtd
```

2. install one or more fdrtd plugins (see e.g. https://github.com/fdrtd/plugin-simon)

3. install and run a server (see e.g. https://github.com/fdrtd/webserver-connexion)
